# MaDaM
macro engine for the game.

# command one line
basic ```^[$0-9A-Z][$0-9A-Z][$0-9A-Z]```
```
KEY
```
with param ```^[$0-9A-Z][$0-9A-Z][$0-9A-Z][ ]```
```
KEY X Y //
```
special math hand ```^$[$0-9A-Z][$0-9A-Z][\=|\%|\+|\-|\>|\<|\?|\!]```
```
$00?X J//
```
# command multi line
```
[[[
...string...on MaDaM
]]]
```
```
{{{
...javascript...on MaDaM
}}}
```
